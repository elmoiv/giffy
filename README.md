# Giffy
**Universal tool to download GIFs from many providers.**

### Current supported providers (14):
  - Twitter
  - Reddit
  - Tumblr
  - Giphy
  - Tenor
  - Imgur
  - 9GAG
  - Sizzle
  - Gfycat
  - GIFbin
  - Reaction GIFs
  - Imgflip
  - Gifer
  - PANA.GIFS

### Testing:
`./run.py` **or** `python run.py`
